Dodatkowe informacje

Elementy liniowe i blokowe

Elementu znajdujące się w body dzielimy na elementy liniowe i blokowe.
Elementy liniowe zajmują dokładnie tyle miejsca ile realnie zajmuje ich zawartość. W jednej linii można umieścić więcej niż jeden taki element. Elementy liniowe to np.: a, em, strong, img.
Elementy blokowe zajmuja zawsze całą dostępną szerokość, nie ważne ile realnie miejsca potrzebują. W jednej linii można umieścić tylko jeden element blokowy. Przykłady: p, ul, ol, h1-h6.
W elementach blokowych można umieszczać elementy liniowe.
Jednak pewne cechy możemy zmieniać w css.

Identyfikatory i klasy

Warto do kontenerów i elementów dodawać identyfikatory i klasy, gdyż pozwalają one odróżniać elementy html od siebie, jak również ułatwiają pracę w css. Dany identyfikator może posiadac tylko jeden element, natomiast daną klasę mozemy nadawać wielu elementom.

CSS

Ogólne informacje

Style do strony można dodawać na 3 sposoby: poprzez atrybut style, poprzez element style dodawany najcześciej w sekcji head, oraz poprzez zewnętrzny plik.css. Najlepiej zawsze dodawać style poprzez zewnętrzny plik i tak będzie to robione w tym przypadku.

Reguły CSS przypisujemy do: elementów HTML, atrybutu id elementu HTML, atrybutu class elementu HTML.
